Панель инструментов OpenStreetMap для <a href="https://github.com/OpenTracksApp/OpenTracks">OpenTracks</a>.
Может использоваться как автономный просмотрщик GPX и местоположений.

Он показывает выбранный путь (трек, трассу) с начальной и конечной точками на карте из <a href="https://openstreetmap.org">OpenStreetMap</a> на основе библиотеки <a href="https://github.com/mapsforge/vtm">Mapsforge VTM</a>.
По умолчанию используется онлайн-карта, но также поддерживаются автономные-карты. С ними нет нужды в мобильной сети.

Карта по умолчанию предоставляется <a href="https://openstreetmap.org">OpenStreetMap.org</a>.
Присоединяйтесь к сообществу и помогите улучшить карту, см. <a href="https://www.openstreetmap.org/fixthemap">OpenStreetMap.org/fixthemap</a>

Пожалуйста, рассмотрите возможность скачивания карты к себе, чтобы уменьшить нагрузку на сервер и сэкономить на вашем мобильном тарифном плане.
Некоторые карты можно найти здесь:
- <a href="https://download.mapsforge.org/">Mapsforge</a>
- <a href="https://www.freizeitkarte-osm.de/android/en/">Freizeitkarte Android</a>
- <a href="https://www.openandromaps.org/en">OpenAndroMaps</a>

Для правильного отображения некоторых карт требуются специальные темы! Их необходимо загрузить и настроить соответствующим образом.

Автономные-карты можно загрузить через меню «Загрузка карты» или посетив указанные выше веб-сайты и нажав на ссылки для скачивания. В некоторых веб-обозревателях (например, Firefox) потребуется долгое нажатие на ссылку, «Открыть ссылку во внешнем приложении», а затем выбрать OSMDashboard.

Скачанные вручную офлайн-карты или карты, предоставленные другим приложениям, можно открыть с вашего устройства Android. Выберите папку в меню «Каталог карт».
